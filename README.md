  1. ApiSong

  2. Crud

  3. Formulario
     1. Funcionalidades completas (validaciones y envío) con capacidad de escalar
  4. Gif app

  5. hooks

  6. Lista de tareas
     1. Un todo list con funcionalidades completas
  7. Modal

  8. Poke app

  9. Select Anidado

  4. npm i @reduxjs/toolkit

  7. npm i animated-css
